# AI Interview System

## Overview

AI Interview System is a web application that simulates technical interviews using AI. It helps users practice interview questions, improve communication skills, and receive AI-generated responses.

## Features

* User Login
* Interactive AI Interview
* AI-powered Question & Answer
* Dashboard
* Responsive User Interface

## Tech Stack

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* Groq API
